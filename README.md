# Travlr Getaways

Full stack travel web application built with the MEAN stack: MongoDB, Express, Angular, and Node.js.

## Features
- Customer-facing Express/HBS website
- Angular admin SPA
- RESTful API for trip data
- MongoDB/Mongoose data storage
- JWT-based admin authentication
- Trip create and update functionality

## Tech Stack
- MongoDB
- Express
- Angular
- Node.js
- Handlebars
- JSON Web Tokens
